Introduction

HTML is the language of the web, the building blocks behind any pages you can see online. everything you see online, being it videos, images, text or even a game, starts with an HTML tag.

Generally speaking, the language of the web serves two mains purposes:

  • - Give semantic meaning to your code
  • - Display the contents of a website

What is HTML?

HTML, which stands for HyperText Markup Language, is not exactly a programming language, but a markup language.

It was invented by Tim Berners-Lee initially for organizing his documents, but later it expanded after he started trying to develop it further and be able to share documents on a network. This eventually culminated on what is know today as the internet.

HTML serves to display contents on a webpage and can be stylized using another language, CSS, and it is given interactivity by a third language, JavaScript. It still developing since its creation and it dominates the web world

What does tag mean?

A tag is a piece of information about the elements contained on a HTML file. It is a piece of text wrapped with "less than"(<) and a "greater than" (>) sinal.

<p>This text is enclosed by a "paragraph" tag;<p>
The <!DOCTYPE html> tag

The <!DOCTYPE html> is not really a tag, it is an instruction to the browser to interpret all the code following it as a certain version of html

It needs always to be added according to the type of html that the document was written in.

<!DOCTYPE html>
code follows this one ...
The <html> tag

The HTML tag is the root of all document, it wraps all the content, both what is display and what is not

It is the beginning of the html document and contains all the other elements.

<html>
...Page content goes here, enclosed by the "html" tag
...
</html>
The <head> tag

The head tag opens the document and comes just after the html tag.

It contains all head elements, most of which are not display and have only instructions for the browser on how to render the page or links for external scripts and stylesheets. The exception to this is the title element, which is displayed on the head of the window tab.

<head>

link, meta, title and script elements go here

</head>
The <meta> tag

The meta tag is non-displayable and contains informations about data, like keywords document type, times pf access and the kind.

It can also be used nowadays for web developers to have control over the viewport.

<meta>

All data information

</meta>
The <title> tag

The title goes inside the head but, differently than other elements located in the head, it is displayed.

It is required in all html documents and displays its name in the brwoser's tab, as well as in search engines.

<title>Page's title</title>
The <body> tag

The body tag wraps all the visual elements of a page.

It contains text, images, videos, and all the contents displayed on any web page.

<body>

Page content displayed in the browser...

</body>
The headings tags

The heading tags are abreviated by the h and decrease in importance and font size from the number 1 to 6. The <h1> tag is the most important and the one that display the text it wraps in a bigger size. The <h6> is the least important and displays the text on a smaller size.

Heading are very important to give semantic meaning when organizing your HTML document. Specially for accessibility reasons when someone might be using a screen header to access the webpage. Thens those tag are mainly used to give structure to the text.

<h1>heading</h1>
<h2>heading</h2>
<h3>heading</h3>
<h4>heading</h4>
<h5>heading</h5>
<h6>heading</h6>
The <header> tag

The header tag defines the header of a page or the top part of a section within it.

It is also import as it gives semantic mening to the html document and helps to make it more accessible.

<header>heading</header>
